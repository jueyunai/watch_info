# 🦔 观猹2025年度报告 —— 融合版策划案

> 「回望365天，看见思维的形状」

---

## 一、项目概述

### 1.1 产品定位
面向所有观猹用户的**年度认知审计工具**。用户输入个人主页即可自动获取2025全年猹评数据，生成精美可视化报告，支持**下载竖屏海报**分享至社交媒体/观猹社区。

### 1.2 核心价值
| 价值维度 | 描述 |
|----------|------|
| **深度** | 不是流水账式的数据汇总，而是一年思维轨迹的深度考古 |
| **温度** | 揭示你自己都没意识到的认知演进，有仪式感的年度仪式 |
| **传播** | 海报分享机制形成社区传播裂变 |

### 1.3 与月报的差异

| 维度 | 月报 | 年报 |
|------|------|------|
| 时间跨度 | 30天 | 365天 |
| 核心问题 | 这个月我在想什么？ | 这一年我变成了什么样的人？ |
| 观点追踪 | 当月快照 | **跨月演进** |
| 情绪分析 | 单点情绪 | **情绪曲线** |
| 输出价值 | 下月行动建议 | 长期方向指引 |

## 三、报告内容框架（6 Part）

### 📊 Part 1: 年度数据全景

**目的**：用数据勾勒一年的输出轮廓

| 指标 | 说明 |
|------|------|
| 📝 猹评总数 | 全年产出的猹评条数 |
| 💬 讨论参与 | 在猹馆发表的帖子数 |
| ✍️ 累计字数 | 估算总字数（万字） |
| 📅 活跃天数 | 有输出的天数 |
| 🔥 高产月份 | 最高产的月份 |
| 😴 低谷月份 | 最沉默的月份 |

**可视化**：月度输出柱状图 + 时间热力图

#### 动态成就标签（根据数据自动匹配）

| 条件 | 标签 |
|------|------|
| 猹评数 ≥ 100 | 🏆 年度高产观猹员 |
| 平均字数 ≥ 500 | 📚 深度测评家 |
| 涉猎产品 ≥ 50 | 🔭 AI产品探索者 |
| 活跃天数 ≥ 200 | ⚡ 每日观猹达人 |
| 猹评数 ≤ 10 但字均 ≥ 800 | 💎 精品主义者 |

---

### 🔥 Part 2: 年度关注力图谱

**目的**：识别你这一年真正在意什么

#### 2.1 高频关键词 Top 10
- 区分「提及频率」和「讨论深度」
- 权重 = 频率 × 上下文字数

#### 2.2 产品品类分布（雷达图/环形图）
- 通用助手 / 图像生成 / 视频创作
- 编程开发 / Agent构建 / 效率工具
- 其他...

#### 2.3 情绪光谱
- **整体基调**：兴奋 / 焦虑 / 批判 / 好奇 / 困惑
- **情绪曲线**：12个月的情绪变化趋势
- **情绪触发点**：最容易让你「上头」的话题

---

### 🧠 Part 3: 认知演进轨迹

**目的**：追踪思维的成长与变化

#### 3.1 年度母题
贯穿全年的 1-3 个核心关注点，以及为什么它们持续吸引你

#### 3.2 观点迭代地图 ✨（亮点模块）
同一话题在不同时期的看法变化，标记「升级」（认知深化）和「转向」（观点改变）

**示例**：
```
🔄 观点迭代：关于「AI产品定价」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3月：「订阅制是AI产品的最优解」
6月：「按量付费可能更适合低频场景」
11月：「混合定价才是终局，关键是价值锚点」

📈 认知轨迹：从非此即彼 → 场景化思考 → 底层逻辑
```

#### 3.3 认知增量清单
- 今年新建立的认知框架
- 新掌握的分析方法论
- 新形成的判断标准

---

### ✨ Part 4: 年度金句与洞见

**目的**：打捞最有价值的思想结晶

#### 4.1 年度金句 Top 5
最具穿透力的原创观点，直接摘录 + 一句话点评

**格式**：
```
💎 年度金句 #1
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
「好的产品不是让用户觉得自己在用工具，
  而是让用户觉得自己变强了」
—— 评 Notion AI，3月15日

💡 洞察价值：点出了AI产品的核心体验标准——赋能感而非工具感
```

#### 4.2 预言验证 ✨（亮点模块）
年初/年中做出的判断 → 年末现实如何？

#### 4.3 意外发现
无心插柳的洞察，当时没在意但现在看很有价值

---

### 🕵️ Part 5: 盲区与反思

**目的**：做自己的镜子，看见看不见的地方

#### 5.1 年度认知矛盾
前后观点冲突的地方——不是「打脸」，而是思维的张力

#### 5.2 被遗忘的钻石
提了一嘴但没深挖的好想法，值得在2026年重新打捞

#### 5.3 思维定势警示
反复出现的认知偏见，如：
- 首因效应：对产品第一印象权重过高
- 技术乐观主义：对AI能力预期偏高
- 完美主义倾向：因追求完美而停滞

---

### 🚀 Part 6: 2026导航系统

**目的**：从回顾中提炼前行的方向

#### 6.1 三个值得深挖的方向
基于今年积累和兴趣点，具体、可执行

#### 6.2 一个「危险」的问题
直击舒适区的灵魂拷问，可能让你轻微不适但必须面对

**示例**：
```
❓ 一个「危险」的问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
「你花了大量时间评论别人的产品，
  但你自己的产品/项目进展如何？」
```

#### 6.3 年度关键词预测（时间胶囊）
基于趋势判断，预测明年可能会关注什么

## 四、海报设计规范

### 4.1 尺寸与布局

| 场景 | 尺寸 | 说明 |

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#3AAF78` | 观猹品牌绿 |
| 深色 | `#2D8B5E` | 渐变/强调 |

| 卡片 | `#1E2A35` | 内容卡片 |

- **标题**：Bebas Neue（观猹官方）

## 六、AI年报提示词（完整版）

```markdown
# 观猹AI洞察年报 2025

## 角色与背景
你是我的「年度认知审计师」，擅长从一整年的碎片化输出中，
提炼出思维演进的脉络、认知成长的轨迹、以及被忽视的盲区。

## 核心任务
请对我在观猹平台2025全年的猹评和讨论进行深度分析，
生成一份《观猹AI洞察年报 2025》。

你的目标是：
1. 穿透表面的「事件流水」，识别贯穿全年的认知母题
2. 追踪同一话题在不同时期的观点演变
3. 发现我自己都没意识到的思维模式和认知盲区
4. 为2026年提供战略级的方向建议

## 输出框架

### 📊 Part 1: 年度数据全景
- 输出总量统计（猹评数、讨论数、估算字数）
- 时间分布特征（高产期、低谷期、节奏规律）
- 一句话概括这一年的输出风格

### 🔥 Part 2: 年度关注力图谱
- 高频关键词 Top 10（区分频率和深度）
- 产品品类分布
- 情绪光谱分析（整体基调 + 情绪触发点）

### 🧠 Part 3: 认知演进轨迹
- 年度母题（1-3个贯穿全年的核心关注点）
- 观点迭代地图（同一话题的看法变化，标注时间节点）
- 认知增量清单（今年新建立的框架/方法论）

### ✨ Part 4: 年度金句与洞见
- 年度金句 Top 5（直接摘录 + 洞察点评）
- 预言验证（年初判断 vs 年末现实）
- 意外发现（被低估的洞察）

### 🕵️ Part 5: 盲区与反思
- 年度认知矛盾（前后观点冲突）
- 被遗忘的钻石（值得重新打捞的想法）
- 思维定势警示（反复出现的认知偏见）

### 🚀 Part 6: 2026展望
- 三个值得深挖的方向
- 一个「危险」的问题（直击舒适区）
- 年度关键词预测

## 分析协议
1. **时间敏感**：注意观点的时间戳，追踪演变而非静态归纳
2. **跨月关联**：寻找不同月份之间的呼应和矛盾
3. **深度优先**：长评权重高于短评，深度讨论权重高于随手吐槽
4. **反直觉检查**：如果结论太显而易见，继续挖掘更隐蔽的联系

## 沟通规则
1. **拒绝流水账**：不要「1月你关注了A，2月你关注了B」
2. **镜像原则**：忠实反馈盲区，不讨好
3. **审慎归纳**：涉及心理推断时，使用「数据暗示...」等客观表述
4. **仪式感**：这是年度报告，语言可以稍微郑重一些

请基于以上框架，开始分析我的2025年度数据。
```
